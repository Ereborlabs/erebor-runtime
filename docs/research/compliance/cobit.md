# COBIT

## What It Is For

COBIT is ISACA's governance framework for enterprise information and technology.
It is useful for Erebor as an IT-governance evidence story, not as a product
claim that Erebor implements COBIT.

## Evidence Companies Need

- governance objectives and management objectives
- process capability evidence
- risk, security, and control practices
- performance and conformance metrics
- audit and improvement evidence

## What Erebor Can Do

Erebor can act as an enforcement and evidence layer for agentic AI inside IT
governance: who owns the agent workflow, what systems it can affect, what
controls apply, what action happened, and what exceptions need review, when
actions are routed through Erebor-controlled execution paths.

## Artifacts Erebor Could Generate

- agent governance responsibility matrix
- runtime conformance evidence report
- policy exception register
- IT governance metrics for AI-agent actions
- audit sample export
- corrective-action tracker

## Do Not Claim

Erebor is not a COBIT implementation, governance operating model, process
capability assessor, enterprise accountability model, or executive metrics
owner. It supports agent runtime controls and evidence.
