# FedRAMP

## What It Is For

FedRAMP is the U.S. federal program for standardized cloud security assessment,
authorization, and continuous monitoring. Erebor fits as supporting runtime
evidence for cloud services or systems in a FedRAMP boundary, not as an
authorization path by itself.

## Evidence Companies Need

- system security plan
- security assessment plan and report
- POA&M
- agency ATO materials
- recurring inventory and vulnerability deliverables
- significant change handling
- incident communication evidence
- annual assessment evidence

## What Erebor Can Do

Erebor can provide runtime control and evidence for AI agents operating inside a
FedRAMP boundary: browsers, terminals, cloud consoles, APIs, SaaS tools, or
production systems. It can capture initiator, policy decision, approval state,
exact action, target system, data exposure, result, and rollback or containment,
when those actions are routed through Erebor-controlled execution paths.

## Artifacts Erebor Could Generate

- AI-agent governed activity ledger
- monthly ConMon appendix for agent runtime
- significant-change package for agent-driven production changes
- incident reconstruction timeline
- shared responsibility matrix for agent controls
- evidence bundle mapped to AC, AU, CM, IA, IR, and CA areas

## Do Not Claim

Erebor is not a FedRAMP authorization, ATO, 3PAO assessment, vulnerability
scanner, SSP/SAP/SAR/POA&M generator, authorization package owner, federal
boundary decision maker, tamper-proof SIEM, rollback system, or full CSPM
platform.
