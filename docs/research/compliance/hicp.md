# HICP

## What It Is For

HICP is healthcare cybersecurity guidance from HHS 405(d). It focuses on
practical practices for protecting the healthcare and public health sector
against common threats.

## Evidence Companies Need

- threat and practice gap assessments
- ransomware playbooks
- identity and access reviews
- asset inventories
- endpoint/email protection evidence
- segmentation evidence
- vulnerability remediation
- incident response exercises
- governance oversight

## What Erebor Can Do

Erebor can treat AI agents as a new operational threat surface. It can block
unsafe SaaS/browser actions, require approval for bulk exports or destructive
commands, constrain agent credentials, log agent-driven changes, and produce
timelines for data-loss or ransomware investigations when those actions are
routed through Erebor-controlled execution paths.

## Artifacts Erebor Could Generate

- HICP AI-agent threat exposure assessment
- runtime evidence mapped to HICP practices
- high-risk SaaS/browser export approval log
- ransomware or data-loss agent activity timeline
- insider-risk agent action report
- agent-governance oversight dashboard

## Do Not Claim

HICP is guidance, not a certification. Erebor does not replace endpoint
protection, SOC operations, IAM, DLP, medical device security, vulnerability
management, or full healthcare governance.
