# HITRUST

## What It Is For

HITRUST provides a harmonized security and privacy control framework plus
assessment and certification programs. It is common in healthcare and maps many
authoritative sources. Its AI-focused assurance products make agent runtime
evidence a credible supporting wedge, but not a certification substitute.

## Evidence Companies Need

- scoped control assessment
- MyCSF evidence collection
- maturity scoring
- assessor validation
- remediation plans
- control inheritance
- third-party risk evidence
- framework-specific reporting

## What Erebor Can Do

Erebor can produce machine-verifiable evidence for controls involving AI-agent
access, privileged workflows, SaaS/browser use, command execution, approvals,
exception handling, and monitoring, when those actions are routed through
Erebor-controlled execution paths.

## Artifacts Erebor Could Generate

- HITRUST AI-agent runtime evidence packet
- access-control and approval matrix
- sampled session logs for assessors
- runtime exception and corrective-action register
- third-party/SaaS agent-use evidence report
- HIPAA/HICP mapping support export

## Do Not Claim

Erebor is not HITRUST certification, a HITRUST assessor, MyCSF, an Insights
Report, CAP management, remediation workflow, or a complete control
implementation. Tie claims to specific controls and evidence.
