# NIST SP 800-171

## What It Is For

NIST SP 800-171 provides security requirements for protecting Controlled
Unclassified Information in nonfederal systems and organizations. The relevant
wedge for Erebor is evidence around governed agent actions in CUI-relevant
workflows, not CUI designation or a complete CUI program.

## Evidence Companies Need

- CUI environment scope
- system/security plans
- implementation evidence
- SP 800-171A assessment evidence
- access records
- audit logs
- incident records
- configuration/change evidence
- supplier evidence
- remediation tracking

## What Erebor Can Do

Erebor can enforce and record governed AI-agent activity across CUI-relevant
systems: approvals before sensitive commands, least-privilege tool access,
browser/SaaS session capture, command provenance, denied actions, and incident
timelines, when those actions are routed through Erebor-controlled execution
paths.

## Artifacts Erebor Could Generate

- CUI agent-operation boundary and data-flow map
- SP 800-171A evidence export for agent-accessed systems
- agent privilege and access review report
- governed command/session audit log
- CUI incident reconstruction and containment timeline
- change-control package for AI-assisted modifications

## Do Not Claim

Erebor is not a complete CUI program, classification/marking/decontrol system,
encryption solution, MFA provider, vulnerability-management program, legal
determination, official assessment package owner, or substitute for assessment.
